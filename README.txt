A flexible WordPress plugin for building directory and listing websites.

== Description ==

All Purpose Directory is a powerful and flexible plugin for creating any type of directory or listing website. Whether you need a business directory, job board, real estate listings, or classifieds site, this plugin provides the foundation you need.

= Key Features =

* **Custom Post Type for Listings** - Full WordPress integration with custom fields, taxonomies, and search.
* **Frontend Submission** - Allow users to submit listings from the frontend with customizable forms.
* **Flexible Field System** - 20+ field types including text, select, checkbox, file upload, and more.
* **Advanced Search & Filtering** - AJAX-powered search with multiple filter types.
* **User Dashboard** - Let users manage their listings from a frontend dashboard.
* **Reviews & Ratings** - Built-in review system with star ratings.
* **Favorites System** - Allow users to save their favorite listings.
* **Fully Customizable** - Override templates in your theme for complete design control.
* **Developer Friendly** - Extensive hooks and filters for customization.

== Installation ==

1. Upload the `all-purpose-directory` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Directory > Settings to configure the plugin.
4. Create your first listing category under Directory > Categories.
5. Start adding listings!

= Minimum Requirements =

* WordPress 6.0 or higher
* PHP 8.0 or higher

== Frequently Asked Questions ==

= Can I customize the listing display? =

Yes! All templates can be overridden in your theme. Simply copy the template from `plugins/all-purpose-directory/templates/` to `your-theme/all-purpose-directory/` and modify as needed.

= Does this plugin work with page builders? =

Yes, the plugin provides shortcodes and blocks that work with all major page builders including Elementor, Beaver Builder, and Gutenberg.

= Can users submit listings from the frontend? =

Yes, frontend submission is a core feature. You can configure which fields appear on the submission form and whether submissions require approval.

= Is the plugin translation ready? =

Yes, the plugin is fully translation ready and includes a POT file for translators.

= Can I import existing listings? =

Yes, the plugin supports CSV and JSON import for bulk listing creation.
